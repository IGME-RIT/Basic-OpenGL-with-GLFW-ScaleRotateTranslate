This tutorial was designed for Visual Studio 2017 / 2019
If the solution does not compile, retarget the solution
to a different version of the Windows SDK. If you do not
have any version of the Windows SDK, it can be installed
from the Visual Studio Installer Tool

Welcome to the SRT Tutorial!
Prerequesites: World matrices, OOP

This tutorial goes more in depth to each type
of 4x4 matrix, showing the actual matrix math
behind it, and how each 4x4 matrix could be 
made by hand. It is not necessary to build them
by hand, but it is helpful to know how the math works
